## Full Content / Transcript

The AI Search Optimization Checklist is the execution layer of the AI search optimization process—a practical way to move from "we should optimize for AI search" to actionable workflow steps.

Key workflow steps:
1. Define the prompts and journeys you actually want to influence—build prompt libraries from sales transcripts, support language, and review data, not keywords alone.
2. Measure current AI presence before changing anything—track prompt coverage, recommendation rate, linked citation rate, comparative win rate, representation accuracy.
3. Diagnose the gap before fixing—map visibility issues to readiness characteristics (Useful, Extractable, Differentiated, Credible, etc.).
4. Make priority pages retrievable and extractable—answer-first structure, standalone paragraphs, FAQ blocks.
5. Improve owned content for decision-support—comparison content, constraint-led pages, original research.
6. Strengthen third-party corroboration—reviews, directories, community mentions.
7. Validate changes with repeated sampling—not single-prompt checks.

The checklist complements the 10 key characteristics framework and 3-layer measurement framework (Presence, Readiness, Business Impact).

Core principle: Optimization should start from observed presence, not from a reflex to create more content.
